🔍 Overview

This project demonstrates an end-to-end data analytics workflow, starting from raw data processing to generating actionable business insights. It involves data cleaning, exploratory data analysis (EDA), SQL-based querying, and building an interactive dashboard for visualization.

The goal of this project is to transform raw data into meaningful insights that support data-driven decision-making.
